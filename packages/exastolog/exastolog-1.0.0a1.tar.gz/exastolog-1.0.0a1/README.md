# pyExaStoLog : a python library for EXAct calculation of STOchastic LOGical models 

## Install
#### From PyPi
    - pip install exastolog
    
#### From your local clone
    - git clone https://github.com/sysbio-curie/pyExaStoLog
    - pip install pyExaStoLog
